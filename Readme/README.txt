======================================================================================   

				== COMMANDES ==



- Aide : Permet d'afficher l'ensemble des commandes

- Inventaire : Permet d'afficher l'inventaire
	
- Aller [Lieu] : Permet d'aller vers le lieu place en parameètre.
           	
- Objectif : Affiche l'objectif actuel (chercher les indices, aller ‡ la sortie)
           	
- Prendre objet : prend un objet. L'equipe si l'equipement en question est vide ou le met dans l'inventaire.
           	
- Utiliser [objet1] : Utilise un objet qui se trouve sur le personnage.
           	
- Ouvrir [Coffre] : Permet d'ouvrir le coffre passÈ en paramètre 
	
- Attaquer [Monstre] : Attaque le monstre se trouvant dans le lieu
           	
- Quitter : Arreête la partie
